# Launch Day Checklist

Use this as a delegable, checkbox-only launch runbook.

- [ ] Push repository and verify public visibility.
- [ ] Run intake preflight: `npm run preflight -- --intake docs/intake-template.json --strict --require-security`.
- [ ] Create release and upload demo-safe assets.
- [ ] Run `npm run demo:refresh` and confirm screenshot freshness.
- [ ] Verify intake link is live and reachable.
- [ ] Publish launch thread with screenshots and single CTA.
- [ ] DM 10 high-fit targets with intake CTA.
- [ ] Log outreach and responses in `docs/kpi-tracker-template.csv`.
- [ ] Schedule follow-up checks for replies and booked calls.
